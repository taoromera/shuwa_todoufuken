# shuwa_todoufuken

A mobile-friendly web quiz app for practicing Japanese Sign Language (手話) for Japan's 47 prefectures (都道府県).

## Quiz modes

- **Video → Name**: Watch a sign-language video and guess the prefecture, then reveal the answer.
- **Name → Video**: See a prefecture name, think of the sign, then reveal the video.

## Videos

Sign-language videos are stored in `videos/` as WebM files named by prefecture romaji (e.g. `hokkaido.webm`, `aichi.webm`).
